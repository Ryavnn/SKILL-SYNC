SkillSync is a specialized freelance marketplace for IT professionals.

Key Features:
- AI-powered job matching
- Verified freelancer credentials
- Freelancer-to-freelancer referral engine
- IT-specific skill taxonomy
- Messaging and milestone payments
- Admin moderation dashboard

Target users:
1. IT freelancers
2. Companies hiring technical talent

Tech stack:
Frontend: Next.js + React
Backend: Node.js / FastAPI
Database: PostgreSQL
AI: OpenAI API
Storage: Cloudflare R2 or S3
Payments: Stripe / Flutterwave
Hosting: AWS or Render
